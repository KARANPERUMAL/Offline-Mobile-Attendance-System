What It Does

Packed with Smart Features

Built to solve a real-world problem — attendance management that doesn't depend on internet connectivity.

📡

100% Offline via LAN

Teacher's phone acts as a local HTTP server on port 8181. Students connect over the same WiFi network — no internet required at all. Perfect for classrooms with poor connectivity.

Core Feature
🔢

OTP-Based Attendance

Teachers generate time-limited OTPs. Students enter the OTP to mark present — prevents proxy attendance.

☁️

Cloud Sync

When internet is available, offline SQLite records sync to the central MySQL server via REST API.

🔐

JWT Authentication

Stateless JWT tokens with role-based access control — Admin, Teacher, and Student roles with different permissions.

📊

Excel Export

Admin can export attendance records as Excel spreadsheets using Apache POI for reports and analysis.

📱

Cross-Platform App

Flutter app runs on both Android and iOS from a single codebase with SQLite local storage.
